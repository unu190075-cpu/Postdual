🚀 PostDual

PostDual is an AI-powered platform that helps creators compare two X (Twitter) posts and determine which one is more likely to perform better before publishing.

✨ Features

- 🤖 AI-powered post comparison
- 📊 Engagement & virality scoring
- 🏆 Automatic winner selection
- 💡 AI suggestions and rewrites
- 🧵 Thread comparison support
- 🖼️ Image/GIF analysis
- 📈 Writing progress analytics
- 📂 Comparison history
- ⭐ Best-performing posts
- 🌙 Dark & Light mode
- 💾 Draft autosave
- 🖼️ Export results as PNG
- 📱 Fully responsive design

🛠️ Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- Supabase
- Framer Motion
- Recharts
